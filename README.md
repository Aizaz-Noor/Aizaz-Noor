**What I'm into right now:** parallel-processing CLI tools, agentic pipelines with human-approval gates (I don't trust fully autonomous agents yet, and I don't think you should either), and figuring out where AI tooling is actually useful vs. where it's just noise.

<details>
<summary><b>💬 Genuinely curious what I'd argue about in a code review — click to see my hot takes</b></summary>
<br/>

- Auto-updaters should be opt-in and loud about what changed, not silent.
- "AI agent does X automatically" is a red flag until I've seen the approval gates.
- Parallel search beats "just add more retries" almost every time.
- A CLI tool with no dry-run flag doesn't fully trust its own logic.

</details>
